About

This project was created as part of nology's pre-course challenges. It was created using HTML, CSS, JavaScript and the Instant-ID.

The aim was to have a web app dedicated to fun while learning more about working with open source API models and practicing API calls.

With OpenAI being behind a paywall, I wanted to explore using other source and stumbled around replicate.com which has a range of exciting Opensourced models to play around with.

Check out replicate: https://replicate.com/

More about instant-id: https://replicate.com/zsxkib/instant-id/examples?input=http&output=json
